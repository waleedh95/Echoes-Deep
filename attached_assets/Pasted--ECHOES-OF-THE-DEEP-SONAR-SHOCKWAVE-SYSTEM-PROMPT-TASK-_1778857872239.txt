## REFERENCE — HOW SUBNAUTICA'S SEAMOTH SONAR WORKS

In Subnautica, when you activate the Seamoth sonar:
- A circular shockwave pulse explodes outward from the submarine in all directions
- The shockwave is a visible expanding ring/sphere of light
- As the shockwave ring passes through geometry (cave walls, rocks, creatures, objects), it "paints" those surfaces with a glowing outline exactly where the wave touched them
- The painted surfaces glow brightly for a moment then slowly fade back to darkness
- The shockwave does NOT illuminate everything at once — it reveals geometry progressively as the ring sweeps through it
- Objects closer to the submarine are revealed FIRST, objects further away are revealed LATER as the ring expands
- The result is a sweeping, cinematic reveal — like dropping a stone in water and watching ripples light up everything they touch

This is NOT a flashlight. This is NOT a flood fill. This is a traveling wave that paints geometry as it passes.

### STEP 3 — GEOMETRY PAINTING (The Core Effect)

This is the most important part. As the shockwave ring passes through each object, paint that object's wireframe edges:

```javascript
function updatePingReveal(ping, deltaTime) {
  
  // Loop through every object in the scene that can be revealed
  scene.traverse((object) => {
    
    if (!object.isRevealable) return; // skip non-world objects
    if (ping.paintedObjects.has(object.uuid)) return; // already painted this ping
    
    // Get distance from ping origin to this object
    const dist = object.position.distanceTo(ping.origin);
    
    // CHECK: Is the shockwave ring currently passing through this object?
    // The ring occupies the space between (radius - thickness) and (radius + thickness)
    const innerEdge = ping.radius - ping.thickness;
    const outerEdge = ping.radius + ping.thickness;
    
    if (dist >= innerEdge && dist <= outerEdge) {
      
      // THE SHOCKWAVE IS TOUCHING THIS OBJECT RIGHT NOW
      // Paint it with a glowing wireframe reveal
      
      revealObject(object, ping);
      ping.paintedObjects.add(object.uuid);
    }
  });
}
```

### STEP 4 — THE REVEAL EFFECT (Glow + Fade)

When an object is "painted" by the shockwave:

```javascript
function revealObject(object, ping) {
  
  // Each world object has two versions:
  // 1. Its solid mesh (always hidden, opacity 0)
  // 2. Its EdgesGeometry wireframe lines (what gets revealed)
  
  const wireframe = object.userData.wireframe; // pre-built EdgesGeometry LineSegments
  
  if (!wireframe) return;
  
  // BURST: Immediately set wireframe to full brightness
  wireframe.material.opacity = 1.0;
  wireframe.material.color.setHex(object.userData.revealColor); // cyan, green, red, etc.
  
  // Store fade timer on this wireframe
  wireframe.userData.fadeTimer = object.userData.fadeDuration || 4.0; // seconds
  wireframe.userData.fading = true;
  
  // Add a brief FLASH — extra bright for 0.1 seconds then settle
  wireframe.material.opacity = 1.5; // over-bright flash (clamped by shader)
  
  setTimeout(() => {
    wireframe.material.opacity = 1.0; // settle to normal bright
  }, 100);
}

// In the main update loop — fade all revealed wireframes back to darkness:
function updateFades(deltaTime) {
  revealedWireframes.forEach((wireframe) => {
    if (wireframe.userData.fading) {
      wireframe.userData.fadeTimer -= deltaTime;
      
      // Opacity follows an ease-out curve (stays bright, then drops fast at end)
      const t = wireframe.userData.fadeTimer / wireframe.userData.totalFadeDuration;
      wireframe.material.opacity = Math.pow(t, 0.4); // ease-out: bright → dim → gone
      
      if (wireframe.userData.fadeTimer <= 0) {
        wireframe.material.opacity = 0;
        wireframe.userData.fading = false;
      }
    }
  });
}
```

### STEP 5 — MULTIPLE SIMULTANEOUS PINGS

Support multiple pings active at the same time (rapid clicking creates overlapping shockwaves):

```javascript
// activePings is an array — can hold multiple pings at once
// Each ping independently expands and paints geometry
// If two pings hit the same object, the second ping RESETS its fade timer (re-brightens it)

// Update loop:
activePings.forEach((ping, index) => {
  ping.radius += ping.speed * deltaTime;
  updatePingReveal(ping, deltaTime);
  
  // Update the visual ring mesh
  ping.ringMesh.geometry.dispose();
  ping.ringMesh.geometry = new THREE.SphereGeometry(ping.radius, 32, 16);
  ping.ringMesh.material.opacity = Math.max(0, 0.5 * (1 - ping.radius / ping.maxRadius));
  
  // Remove completed pings
  if (ping.radius >= ping.maxRadius) {
    scene.remove(ping.ringMesh);
    activePings.splice(index, 1);
  }
});
```

## SCENE SETUP — HOW TO BUILD REVEALABLE OBJECTS

Every world object must be set up with both a hidden mesh and a wireframe overlay:

```javascript
function createRevealableObject(geometry, position, revealColor, fadeDuration) {
  
  // The solid mesh — hidden, used only for collision
  const solidMesh = new THREE.Mesh(geometry, new THREE.MeshBasicMaterial({ visible: false }));
  solidMesh.position.copy(position);
  solidMesh.isRevealable = false; // solid mesh is NOT directly revealed
  
  // The wireframe — what gets lit up by the sonar
  const edges = new THREE.EdgesGeometry(geometry);
  const wireMaterial = new THREE.LineBasicMaterial({
    color: revealColor,
    transparent: true,
    opacity: 0,              // INVISIBLE by default
    linewidth: 2,
    blending: THREE.AdditiveBlending  // makes lines glow/bloom naturally
  });
  const wireframe = new THREE.LineSegments(edges, wireMaterial);
  wireframe.position.copy(position);
  wireframe.isRevealable = true;    // THIS gets painted by sonar
  wireframe.userData.revealColor = revealColor;
  wireframe.userData.fadeDuration = fadeDuration;
  wireframe.userData.totalFadeDuration = fadeDuration;
  
  // Link them
  solidMesh.userData.wireframe = wireframe;
  wireframe.userData.solidMesh = solidMesh;
  
  scene.add(solidMesh);
  scene.add(wireframe);
  
  return { solidMesh, wireframe };
}

// USAGE EXAMPLES:
createRevealableObject(new THREE.BoxGeometry(10,10,10), wallPos, 0x00FFFF, 4.0);      // cave wall
createRevealableObject(new THREE.CapsuleGeometry(2,6), podPos, 0x00FF88, 5.0);        // lifepod
createRevealableObject(enemyGeometry, enemyPos, 0xFF3333, 1.5);                        // enemy
```
